📦 Lost Products Portal

A simple Lost & Found web portal that allows users to report, search, and manage lost or found items such as ID cards, documents, notebooks, or other personal belongings.

This project helps connect people who lost an item with those who found it, making the recovery process easier and faster.

🚀 Features

✅ Report lost items
✅ Report found items
✅ Search functionality for items
✅ Clean and responsive UI
✅ Easy item listing and viewing
✅ Works on desktop and mobile browsers
